# Генератор розкладки продуктів

Десктопний додаток для генерації PDF-таблиць розкладки продуктів з XLSX-файлів.  
Три режими: **розкладка по днях**, **зведена таблиця за тиждень**, **масштабування накладної**.

---

## Windows

### 1. Встановити Python

Завантажити інсталятор з [python.org](https://www.python.org/downloads/) і запустити.

> ⚠️ На першому екрані обов'язково поставити галочку **"Add Python to PATH"**

### 2. Завантажити програму

На сторінці репозиторію натиснути **Code → Download ZIP** і розпакувати в зручне місце (наприклад `C:\Кухня`).

### 3. Встановити залежності

Двічі клікнути на файл **`setup.bat`** у папці програми.  
Скрипт автоматично завантажить GTK і встановить усі Python-пакети.

> ⚠️ Потрібне підключення до інтернету. Під час встановлення GTK може відкритись вікно інсталятора — просто дочекайтесь.

### 4. Запустити

Двічі клікнути на файл **`run.bat`** у папці програми.

---

## macOS

### 1. Встановити Python

Завантажити інсталятор з [python.org](https://www.python.org/downloads/) і встановити.

> Рекомендується саме python.org, а не Homebrew — краща сумісність з tkinter на macOS.

### 2. Завантажити програму

На сторінці репозиторію натиснути **Code → Download ZIP** і розпакувати (наприклад в `~/Desktop/Кухня`).

### 3. Встановити залежності

Відкрити термінал і виконати:

```bash
pip3 install openpyxl num2words weasyprint
```

### 4. Запустити

```bash
cd ~/Desktop/Кухня
python3 app.py
```

---

## Використання

| Вкладка | Що робить |
|---|---|
| **По днях** | XLSX → один PDF (A3, альбомний) з 7 аркушами розкладки по днях |
| **Зведена** | XLSX → один PDF з таблицею «Усього за тиждень» |
| **Накладна** | Масштабує xlsx-накладну на задану кількість осіб |

**Дата початку тижня** заповнюється автоматично з імені XLSX-файлу, якщо воно містить дати у форматі `06.04-12.04.2026`.
